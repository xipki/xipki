Download and copy the BouncyCastle jars here before calling `prepare.sh`.

**DO NOT** rename the jars.

You can choose to use FIPS variant, LTS variant, or standard variant
(but not mixed)

- FIPS variant
  - bc-fips-<version>.jar
  - bcpqc-fips-<version>.jar
  - bcutil-fips-<version>.jar
  - bcpkix-fips-<version>.jar
- LTS variant
  - bcprov-lts8on-<version>.jar
  - bcutil-lts8on-<version>.jar
  - bcpkix-lts8on-<version>.jar
- Standard variant
  - bcprov-jdk8on-<version>.jar
  - bcutil-jdk8on-<version>.jar
  - bcpkix-jdk8on-<version>.jar
